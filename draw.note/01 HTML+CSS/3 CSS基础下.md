+   常见的单位
    ```
        - px 像素

        - em 基于字体大小的倍数

        - rem html字体大小的倍数

        - vw 视口的宽度

        - wh 视口的高度

        - % 百分比
            a) 如果是宽高百分比是基于外层标签的宽高
            b) 行高百分比是基于字体大小的倍数
    ``` 
+   CSS常用属性
    ```
        1. width/height: 宽高,块和行内块标签有效

        2. background-color: 背景颜色

        3. color: 字体颜色

        4. font-size: 字体大小

        5. text-align: 文本水平对齐方式

        6. text-indent: 首行缩进

        7. font-family: 字体类型(黑体,宋体,微软雅黑等等)

        8. font-weight: 字体加粗,bloder blod light lighter normal(默认)

        9. font-style: italic斜体 /normal(默认值)

        10. line-height: 行高
            - 当行高等于盒子高度,字体在盒子中垂直居中
            - 如果没有给盒子写了行高没有写高度,高度会等于行高的高度;
    ```
+   font简写
    ```
        - font: font-style font-variant font-weight font-size/line-height/font-family;

        - 简写需要有font-size和font-family才会生效;
    ```
+   background属性
    ``` 
        1. background-color: 背景颜色;

        2. background-image: 背景图片url('');

        3. background-repeat: 平铺方式
            - x轴从左至右,y轴从上到下;
            - repeat(默认值) no-repeat repeat-x repeat-y;
        
        4. background-position : 图片位置
        
    ```
