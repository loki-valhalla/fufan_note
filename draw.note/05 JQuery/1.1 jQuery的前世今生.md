+   JS框架库

    ```
        - js框架库就是普通的js文件,这些文件中封装了很多函数和兼容的代码;
    ```

+   什么是jQuery?
    ```
        - jQuery就是一个js框架库;

        - jQuery的作用是为了简化开发;

        - js中有些功能jQuery没有实现,仍然需要使用原生的js才能实现;
    ```

+   为什么需要jQuery?
    ```
        - DOM中有些简单的问题需要大量的代码;

        - DOM中的兼容问题很多;

        - DOM中代码的容错性很差;

        - DOM中多个window.onload会被覆盖;

    ```

+   jQuery的特点?
    ```
        - write less do more;

        - 解决js兼容问题(不包括css);

        - 链式编程,隐式迭代;

        - 使用简单,插件丰富,开源,免费;
    ```
