+   显示隐藏动画效果
    ```
        - show()/ hide()
            a) 参数: 第一个参数是执行的时间单位毫秒, 第二个参数是动画结束后执行的回调函数
        
        - toggele()
            a) 作用: 让显示的隐藏,让隐藏的显示
    ```

+   展开和收起动画效果
    ```
        - slideDown(1000,function(){})
            a) 展开

        - slideUp()
            a) 收起
        
        - slideToggle()
            a) 切换
    ```

+   淡入淡出
    ```
        fadeIn() 

        fadeOut()

        fadeToggle()
        
        fadeTo()

    ```
+   动画队列:
    ```
        - 执行动画的时候,如果上一次触发的动画没有执行结束,第二个事件的动画会默认排到后面执行;

        - stop() 停止正在执行的动画
            a)  $().stop() --> 停止当前执行的执行下一个;
           
            b) stop的参数: (true) 立即停止当前的和后续的动画;

            c) stop的参数: (false,true) 立即完成当前的,继续执行后续;

            d) stop的参数: (true,true) 立即完成当前的,并且停止后续的;

    ```

+   自定义动画
    ```
        - 在开发中jQuery封装好的动画满足不了我们的需求也可以自定义动画;

        - animate中传递四个参数:
            a) 参数1 : 传递一个对象要修改的目标位置,宽高等属性;

            b) 宽高等也可以设置为追加值,使用字符串追加,例:'+=200'; (在原来的基础上追加200)

            c) 关键字:'hide'隐藏宽度,'toggle'宽度切换

            d) 参数2: 动画执行的时间

            e) 参数3: 缓动(swing)还是(linear)匀速动画,默认是缓动;

            f) 参数4: 回调函数,在动画结束后执行;
        
        - 自定义动画: animate({},1000);

        - 多个动画可以链式执行

        - delay() 在执行完动画后使用delay() 会延迟之后再执行;
        例: animate({},1000).delay(1000).animate({},1000)
        
    ```
