+   jQuery之静态方法的原理
    ```
        - 以JS中的静态对象Math为例:

            1. 以字面量的形式创建对象:
               var Math = {
                   PI:3.1415926535;
               };

            2. 以构造函数的形式创建:
                function Math(){ ... }
                // 作为构造函数的直接属性存在;
                Math.PI = 3.1415926535;

    ```
